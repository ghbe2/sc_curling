# PROTO1_INSTRUCTIONS.md — テイクアウト手触り検証プロトタイプ

## これは何か

過去のSPEC.md / SPEC_V2.md、game_m1.html / game_v2_m1.html はすべて別方向(タップ全停止型、常時傾く皿型)のもので、**今回は無関係**。これらは参照しなくてよい(衝突解決のコードだけ流用したければ見てもよいが、姿勢計算・傾き計算・接触判定は今回一切不要)。

今回作るのはゲームではなく、**「投げる→カーブする→当てる→弾き飛ばす→連鎖する」という1アクションが単体で楽しいかどうかを確認するための検証用プロトタイプ**。スコア・ライフ・ゲームオーバー・キャラの個性・人質ギミックは一切含めない。それらは「楽しい」が確認できてから別途指示する。

新規ファイル `proto1_takeout.html` として作成すること(既存ファイルの上書き・拡張はしない)。

## 画面構成

- 2D真上視点。円形の盤面(仮固定でよい、形状の検討は後回し)。
- 盤面の外に出たピースは画面から消える(弾き出し成功、演出は縮小+フェード程度でよい)。
- 盤面外の余白に「リセット」ボタンと、後述のテストシナリオ切り替えボタンを配置する。

## 操作

- 画面上でドラッグ→リリースの1ジェスチャーで、自分のピース(円)を盤面に向けて投げる。
- ドラッグの軌跡が曲がっているほど、投げたピースがカーブする(ポケモンGOのカーブボールと同じ考え方)。カーブ量はドラッグ軌跡の曲率から1つの数値(-1.0〜1.0程度)に変換し、それを横方向の加速度として物理側に渡す。
- ドラッグの距離・速度で初速(威力)を決める。
- 極端に短いドラッグ(タップに近い)は投げない(誤操作防止の閾値をCONFIGに用意する)。

## 物理モデル(簡易でよい、姿勢計算・回転行列は禁止)

- 全ピースは同一の円(半径・質量は全部共通の定数)。
- 移動中は摩擦により等加速度的に減速し、必ず停止する。
- カーブは「進行方向に対して常に一定の横加速度がかかる」というだけの単純なモデルでよい。カーブ量が大きいほど横加速度が強い。
- ピース同士は円の弾性衝突として処理する。連鎖(玉突き)が自然に起きること。
- 衝突で弾かれたピースが盤面の縁を越える速度を持っていれば場外(消滅)、遅ければ縁で止まる。

## テストシナリオ

画面上のボタンで以下を切り替えられるようにする。数値配置は目安でよい。

1. **単体ターゲット**:盤面中央付近にターゲット1個のみ配置。狙い撃ちの手触り確認用。
2. **密集ターゲット**:盤面のどこかに3〜5個を近接配置。連鎖テイクアウトの手触り確認用。
3. **壁越え**:プレイヤーの投げる位置とターゲットの間に1個の壁ピースを配置。カーブで避けて当てる手触り確認用。
4. **フリー配置**:ランダムな位置に5〜8個配置。「リセット」ボタンで毎回再配置される。

各シナリオ切り替え時、盤面上のピースは全てクリアしてから再配置する。

## CONFIG(ファイル冒頭に集約、プレイテスト中に直接書き換える前提)

```js
const CONFIG = {
  BOARD_RADIUS: 320,
  FRICTION: 0.985,       // 1フレームごとの速度減衰率
  CURVE_STRENGTH: 0.15,  // カーブ量→横加速度の変換係数
  RESTITUTION: 0.6,      // ピース間反発係数
  MIN_SWIPE_DIST: 20,    // これ未満のドラッグは投げない
  THROW_POWER_SCALE: 1.0,// ドラッグ速度→初速の変換係数
  PIECE_RADIUS: 24,
  SETTLE_SPEED: 0.05,    // これ未満の速度になったら停止扱い
};
```

## やらないこと(明記して念押し)

- スコア表示、得点計算
- ライフ・ゲームオーバー・エンドレス進行
- ピースごとの重さ/カーブ効き/耳・角などの個性差(全ピース同一)
- 人質・救出ギミック、キャラ固有の動作パターン
- グラフィック(単色円+回転が分かる程度の模様でよい。パーツ描画は禁止)
- 外部ライブラリ・画像・音声

## 納品時に確認してほしいこと

- カーブがスワイプの軌跡通りに曲がる感覚があるか(直感的かどうか)
- 密集ターゲットで連鎖(複数同時テイクアウト)が実際に起きるか、起きたときに見た目として分かりやすいか
- 壁越えシナリオで「避けて当てる」ができるか、できたときに達成感があるか
- 現状のCONFIG値で気持ちよさが出ているか、出ていなければどの値が怪しいか(所感でよいので一言添える)
